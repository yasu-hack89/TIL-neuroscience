# Day 3 — Ch.3 Signaling across synapses

## シナプス伝達の流れ
活動電位が軸索末端に到達
→ Ca²⁺チャネル開口・Ca²⁺流入
→ 小胞がシナプス間隙に神経伝達物質を放出

Ca²⁺の流入が引き金

## 後シナプスの受容体

### イオノトロピック受容体
- 受容体自体がチャネル → 結合で即開口
- 速い（ミリ秒単位）
- 例：AMPA・NMDA・GABA-A

### メタボトロピック受容体
- 結合 → Gタンパク質活性化 → セカンドメッセンジャー産生 → チャネル開口or遺伝子発現変化
- 遅いが増幅効果が大きい
- 例：ドーパミン・セロトニン・ムスカリン性ACh受容体
- 多くの精神科薬・麻酔薬の標的（GPCR）

## EPSPとIPSP
- EPSP（興奮性）：膜電位をプラス方向 → 発火しやすくなる
- IPSP（抑制性）：膜電位をマイナス方向 → 発火しにくくなる
- シナプス統合：EPSPとIPSPの足し引きで閾値を超えるか決まる
